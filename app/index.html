What We Do

The Ministry of Energy and Mineral Development, in partnership with NITA-U and the Uganda Revenue Authority, operates MafutaWatch Uganda to eliminate information asymmetry in the fuel sector. Our platform protects consumers from un-consented price manipulation by providing real-time price transparency, automated fraud detection, and direct government accountability channels.

Our reports track more than 1,400 retail stations daily.

Across all 15 districts and 4 administrative regions of Uganda.

🔌 API Payload Specification — Fraud Moderation
POST /v2/fraud/action
{
  "action": "approve" | "dismiss" | "throttle",
  "flagId": "integer — fraud queue entry ID",
  "moderatorId": "string — admin user ID",
  "reason": "string — optional moderation note",
  "blacklistUser": boolean,   // only when action=throttle
  "blacklistDuration": "string" // e.g. "permanent", "30d"
}
📡 API Payload Specification — Station Operator Auth
PATCH /v2/operators/authorize
{
  "applicationId": "integer — KYC application ID",
  "status": "approved" | "rejected",
  "tokenName": "string — for auto-generated API token",
  "rateLimit": "string — e.g. '500 req/min'",
  "authorizedBy": "string — admin user ID"
}
